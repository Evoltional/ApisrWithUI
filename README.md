# APISR - Advanced Python Image Super-Resolution

一个基于深度学习的视频超分辨率处理工具，支持多种先进的超分辨率模型，具有重复帧检测、进度恢复等功能。
## 前置条件

Python 3.8+

CUDA兼容的GPU（推荐）

FFmpeg

## 功能特性

🎯 支持多种超分辨率模型：GRL、DAT、RRDB、CUNET

🚀 智能重复帧检测：支持哈希和SSIM双重检测

📊 历史帧缓存：可配置历史帧数量，提高检测准确性

💾 进度恢复：支持暂停/继续，可保存和恢复处理进度

⚡ 性能优化：支持FP16推理，GPU内存管理

🎛️ 灵活配置：丰富的参数选项，满足不同需求

📝 详细日志：完整的处理过程记录

## 运行方法

```
python ApisrUI.py
```

## 配置参数

选择输入视频文件

设置输出目录

选择模型和缩放因子

调整其他参数（可选）

## 使用说明

### 模型参数

模型选择：GRL/DAT/RRDB/CUNET

缩放因子：2x/4x（取决于模型）

分段时长：视频分段处理的时长（秒）

### 性能设置
批处理大小：GPU批处理大小（建议1-2）

瓦片大小：处理时的瓦片大小

FP16推理：启用半精度推理加速

### 重复帧检测
启用检测：开启重复帧检测

哈希检测：基于感知哈希的检测

SSIM检测：基于结构相似性的检测

历史帧数量：用于比较的历史帧数（1-100）


### 补充
首次运行时会自动创建配置文件 apisr_config.json

临时文件以视频文件名命名，便于管理

处理过程中会自动保存进度，支持断点续传

建议使用GPU以获得最佳性能
